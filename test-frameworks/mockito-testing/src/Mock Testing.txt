Mock Testing: Use mock objects to isolate the code to be tested
在测试过程中，将实际要输出的逻辑进行替换，使用假的结果来验证测试
在单元测试的时候，使用Mock Object去模拟和隔离依赖性的问题

Mock frameworks:
- Mockito
- Powermock
- EasyMock
- JMockIt
- JMock
- MockRunner

按照测试的层面划分:
- Test offline / online
- Test unitaire / composant / package

按照测试的细节划分:
- Unit Test: 直接在JVM中完成, 测试最小的功能单元
- Integration Test: 启动Service服务, 能够测试一系列的功能
- Component Test: 测试不同服务/不同组之间的依赖
- Package Test: 测试软件集成的不同包
- End2End test: 用户应用层面的端到端测试
